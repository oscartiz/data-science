# Data Science Projects Repository

This repository serves as a monorepo containing various data science and machine learning projects.

## Projects Directory

- **[Omnichannel Equilibrium](./omnichannel_equilibrium/)**: A prescriptive analytics system using Double Machine Learning and Multi-Agent Reinforcement Learning (MARL).
- **[Tortilla Prices Analysis](./tortilla_prices/)**: A multidimensional analysis of tortilla price trends in Mexico, featuring a custom "Tortilla Index" and ARIMA forecasting.
- **[Job Salary Prediction](./job_salary_prediction/)**: A Deep Neural Network (DNN) implementation in PyTorch to predict salaries based on professional features, optimized via Grid Search.
